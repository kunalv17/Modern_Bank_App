Save settings to profile

Specify a profile name to store current settings of archiving dialog. You will be able to quickly restore them by pressing "Profiles" button in archiving dialog and selecting the appropriate profile name in menu. Archive name and password are not stored in the profile.

If you enable "Set profile as default" option, settings stored in such profile will be automatically read when opening the archiving dialog. Only one profile can be specified as default, so if you set this option for a new profile, RAR will reset it for any other profile which had the default status.
