Settings

Settings dialog can be opened from Options menu.

"About" and "Help" toolbar buttons

"About" displays version and copyright information about RAR. "Help" shows this help screen.

Interface, File list, Paths, Other

Separate screens with interface, file list, paths and other options.

Sort file list

Select the sort mode for file list.

Names encoding

Encoding for non-English file names in non-Unicode archives. Most of encodings are available beginning from Android 4.4.

Advertisements

Advertisement settings such as personalized or non-personalized ads. Might be unavailable depending on presence of advertisements in current RAR version, user's region and state of "Remove advertisements" option purchase.
