Physical keyboard and mouse

RAR supports following additional shortcuts for physical keyboard connected to Android device:

Alt+A
Add to archive
Alt+E
Extract
Ctrl+A
Select all files
Ctrl+C
Copy
Ctrl+V
Paste
Ctrl+X
Cut
Del
Delete files
Ins
Invert file selection
Shift+Up, Shift+Down
Select files

If physical mouse is connected and device supports right clicks, such right button click on a file displays the context menu.
