Organize profiles

Tap on checkboxes or profile names to select an item. You can select one or multiple profiles. When all modifications are done, press OK button to store them.

Following commands are available:

Select all

Select all compression profiles in the list. If all profiles are already selected, this command deselects them.

Delete

Delete selected profiles.

Set as default

Assign 'default' status to selected profile. Such profile will be used to set initial parameters in archiving dialog. If several profiles are selected, first of them gets the default status.
