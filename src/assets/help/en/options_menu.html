Options menu

This menu is displayed when pressing the overflow icon in RAR top right corner or with device Menu button, if available.

Test archived files

Verify archived data and report errors if found.

Repair archive

Attempt to repair selected or opened archive.

Benchmark

Benchmark device performance.

Send

Send selected files to other applications. It can be used both for archived and usual files. If activated inside of archive and no files are selected, the entire archive is sent. If some archived files are selected, they are unpacked to temporary folder and then sent.

Information

Display information about selected files and folders if invoked outside of archive. Display archive information if invoked inside of archive.

Remove advertisements

You can pay to remove advertisements in RAR. After purchase is done, this menu item becomes unavailable. It is also unavailable if current RAR version does not include advertisements.

Settings

Open Settings dialog to configure RAR interface and other parameters.

Help

Display help contents.

Exit

Close RAR.
