Browse for file (archive)

This dialog can return a full file name including path, update path to file preserving name or return folder name.

Tap on folder names to open them, so you can locate a folder containing required files. If you found a needed file or archive, tap on its name.

If you need only folder name, but not file name, press "Folder only" button. Its behavior may vary in different RAR modules. When called from archiving dialog, it updates the path component of archive name preserving the original name component. In other RAR modules it may return the full folder name not preserving the original name.

Use navigation panel to select the current folder from list of favorite folders. Long press on any file or folder name to open the context menu, which allows to create a new folder.
