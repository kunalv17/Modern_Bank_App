Organize favorites

Tap on checkboxes or favorite names to select an item. You can select one or multiple favorites. When all modifications are done, press OK button to store them.

Following commands are available:

Select all

Select all favorites in the list. If all favorites are already selected, this command deselects them.

Delete

Delete selected favorites.

Edit

Modify parameters of selected favorite. If several favorites are selected, only first of them will be edited.
