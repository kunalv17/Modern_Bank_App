Enter password

Specify a password to encrypt files when archiving or to decrypt an encrypted file when extracting.

If you archive files with password, for maximum security it is recommended to:

  • use RAR format, because it utilizes a stronger encryption algorithm;
  • avoid passwords shorter than 8 characters;
  • avoid common words as passwords.

Remember your passwords. RAR encryption does not contain any backdoors. RAR developers are not able to help you to restore your passwords.

Show password

Hide or show the password as you type.

Encrypt file names

This option is available only when creating RAR format archives. By default, if this option is off, only file data is encrypted. File names, sizes and dates are still visible without password. When enabling this option, both file data and file names, sizes, dates are encrypted, so password is required even to see the list of files contained in the archive.

Remember for <N> minutes

This option is available only when extracting, testing or browsing archives. It allows to remember an entered password and apply to all extraction commands during specified number of minutes. It might be convenient when unpacking multiple archives with same password. Password is used until specified time expires or data error is found or RAR is closed with Back button.
