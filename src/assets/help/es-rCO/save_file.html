Guardar archivo

Toque en los nombres de las carpetas para abrirlas, de modo que pueda localizar una carpeta para almacenar el archivo. También puede editar su nombre. Presione "Aceptar" para guardar el archivo en la carpeta abierta actualmente.

Utilice el panel de navegación para seleccionar la carpeta actual de la lista de carpetas favoritas. Presione prolongadamente sobre cualquier nombre de archivo o carpeta para abrir el menú contextual, que permite crear una nueva carpeta.
