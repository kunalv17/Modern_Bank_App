Opciones de compresión de archivos: Comentario

Cargar un comentario desde archivo

Especifica una ruta de archivo para añadir su contenido como comentario del archivo comprimido. Dicho archivo debe tener el formato de texto UTF-8. Utilice "Examinar" para localizar un archivo.

Escribir un comentario manualmente

Se escribe el texto para añadirlo como comentario de archivo comprimido. Se permiten comentarios multilínea.

Puede cargar un comentario al archivo o introducirlo manualmente. Utilizar ambas opciones juntas no es posible.
