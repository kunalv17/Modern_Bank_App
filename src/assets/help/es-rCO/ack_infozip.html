Código fuente Info-ZIP

RAR incorpora el código fuente del grupo Info-ZIP, que se utiliza para manipular archivos ZIP. El código fuente original de Info-ZIP está disponible de forma gratuita en ftp://ftp.info-zip.org/pub/infozip/ y www.info-zip.org.
