Menú de opciones

Este menú se muestra cuando se pulsa el ícono de desbordamiento de la esquina superior derecha de RAR o con el botón de Menú del dispositivo, si está disponible.

Comprobar archivos comprimidos

Verifica los datos comprimidos y reporta si hay errores.

Reparar archivo

Intenta reparar el archivo comprimido seleccionado o abierto.

Rendimiento

Evalua el rendimiento del dispositivo.

Enviar

Envía los archivos seleccionados a otras aplicaciones. Se puede utilizar tanto para los archivos comprimidos como para los habituales. Si se activa dentro del comprimido y no se seleccionan archivos, se envía todo. Si se seleccionan algunos archivos comprimidos, se descomprimen en una carpeta temporal y luego se envían.

Información

Muestra la información de los archivos seleccionados y carpetas si se utiliza fuera de los comprimidos. Muestra la información del archivo comprimido si se utiliza dentro de un archivo.

Quitar anuncios

Se puede pagar para quitar los anuncios en RAR. Una vez realizada la compra, este elemento del menú no estará disponible. Tampoco está disponible si la versión actual de RAR no incluye anuncios.

Ajustes

Abre el Diálogo de ajustes para configurar la interfaz RAR y otros parámetros.

Ayuda

Muestra el contenido de la ayuda.

Salir

Cierra RAR.
