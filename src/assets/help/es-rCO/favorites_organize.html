Organizar favoritos

Presione las casillas de verificación o los nombres favoritos para seleccionar un elemento. Puede seleccionar uno o varios favoritos. Cuando haya realizado todas las modificaciones, presione Aceptar para guardarlas.

Los siguientes comandos están disponibles:

Seleccionar todo

Selecciona todos los favoritos de la lista. Si todos los favoritos ya están seleccionados, este comando los deselecciona.

Eliminar

Elimina los favoritos seleccionados.

Editar

Modifica los parámetros del favorito seleccionado. Si se seleccionan varios favoritos, sólo se editará el primero de ellos.
