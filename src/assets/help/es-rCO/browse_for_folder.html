Buscar carpeta

Presione en los nombres de las carpetas para abrirlas, de modo que pueda localizar la carpeta necesaria. Abra esta carpeta y presione "Aceptar" para devolverle el nombre.

Utilice el panel de navegación para seleccionar la carpeta actual de la lista de carpetas favoritas. Presione por largo tiempo en cualquier nombre de archivo o carpeta para abrir el menú contextual, que permite crear una nueva carpeta.
