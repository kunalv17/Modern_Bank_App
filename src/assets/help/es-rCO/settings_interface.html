Interfaz

Tema

Seleccione el tema de la interfaz oscuro, claro o negro.

Sonido

Emite un sonido de pitido después de completar una operación. No se emite un aviso sonoro si la operación se completó rápidamente.

Ocultar barra de notificación

Oculta la barra de notificación del sistema en la parte superior de la pantalla.

Tamaño de letra

Cambia el tamaño de la fuente para la lista de archivos, la barra de estado y algunas otras partes de la interfaz. Elija valores positivos para la fuente más grande y negativos para la más pequeña.
