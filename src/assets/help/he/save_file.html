שמירת קובץ

הקש על שמות תיקיות כדי לפתוח תיקיות, על מנת לאתר תיקיה לאחסון הקובץ. לחץ "אישור" כדי לשמור את הקובץ בתיקיה הפתוחה הנוכחית.

ניתן גם להשתמש בחלונית הניווט כדי לבחור בתיקיה לשמירה מרשימת התיקיות המועדפות. לחץ לחיצה ארוכה על כל שם קובץ או תיקיה כדי לפתוח את תפריט ההקשר, המאפשר ליצור תיקיה חדשה.
