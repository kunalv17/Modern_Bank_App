Ajouter aux favoris

Indiquez un nom et un emplacement pour créer une nouvelle entrée dans la liste des favoris ou effectuer un changement. Vous pouvez accepter les valeurs par défaut proposées ou les modifier selon votre besoin. La saisie dans le champ "Nom" s'affichera dans le sous-menu "Favoris" du panneau de navigation. L'emplacement doit contenir le chemin complet vers le dossier existant.
