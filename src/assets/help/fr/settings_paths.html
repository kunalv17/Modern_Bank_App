Destinations

Dossier d'extraction par défaut

Spécifiez un dossier qui sera proposé par défaut comme destination pour les fichiers extraits dans la boîte de dialogue d'extraction et le menu contextuel.

Utilisation du dossier par défaut

Définissez quand utiliser le dossier d'extraction par défaut. Les choix possibles sont «Jamais», «A l'intérieur du dossier temporaire uniquement" et "Toujours". Tout le contenu du dossier temporaire Android/data/com.rarlab.rar est supprimé lors de la désinstallation de l'application RAR et cela peut conduire à la perte de données s'il contient déjà des fichiers décompressés. Par défaut cette option est donc réglée sur "A l'intérieur du dossier temporaire uniquement» et, si les archives sont stockées dans le dossier temporaire, ce qui peut arriver si elles proviennent d'une autre application, RAR propose automatiquement d'extraire les archives en question dans le dossier d'extraction par défaut en dehors du dossier temporaire.

Ajouter un nom d'archive

Permet de contrôler si le nom d'une archive sans extension est ajoutée à la destination initialement proposé dans la boîte de dialogue d'extraction. Cela n'affecte pas la commande d'extraction dans un dossier depuis le menu contextuel

Historique d'extraction

Si activée, les chemins de destination dans la boîte de dialogue d'extraction sont ajoutés à l'historique et peuvent être sélectionnés ultérieurement avec le bouton "Chemin de destination" à partir de cette boîte de dialogue. Les chemins de destination sélectionnés directement depuis le menu contextuel ne sont pas ajoutés à l'historique.

Dossier de démarrage

Sélectionnez un dossier de démarrage après la mise en route de RAR. Par défaut, c'est le dossier racine de la mémoire du périphérique.
