Enregistrer les paramètres au profil

Indiquez un nom de profil pour stocker les paramètres actuels de la boîte de dialogue d'archivage. Vous serez en mesure de les restaurer rapidement en appuyant sur le bouton "Profils" dans la boîte de dialogue d'archivage et en sélectionnant le nom du profil approprié dans le menu. Le nom de l'archive et son mot de passe ne sont pas enregistrés dans le profil.

Si vous activez l'option "Définir le profil par défaut", les paramètres enregistrés dans ce profil seront lu automatiquement lorsque vous ouvrez la boîte de dialogue d'archivage. Un seul profil par défaut peut être spécifié, donc si vous définissez cette option pour un nouveau profil, RAR enlèvera ce statut à tout autre profil qui était défini comme profil par défaut.
