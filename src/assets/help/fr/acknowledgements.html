Remerciements

RAR intègre le code du groupe Info-ZIP permettant de gérer les archives ZIP.

RAR4 et l'algorithme de décompression ZIP PPMD sont basés sur le code source public de Dmitry Shkarin et pour PPMd de Dmitry Subbotin.

Le chiffrement de RAR comprend des extraits de code de Szymon Stefanek , des mises en application AES de Brian Gladman et le code source SHA-1 de Steve Reid.

RAR utilise la fonction CRC32 basée sur l'algorithme Intel Slicing-by-8.

Les archives RAR peuvent éventuellement inclure un hash BLAKE2sp, conçu par Jean-Philippe Aumasson, Samuel Neves, Zooko Wilcox-O'Hearn et Christian Winnerlein.

De nombreux conseils utiles fouris par Alexander Khoroshev et Bulat Ziganshin nous ont permis d'accroître significativement la vitesse et le taux de compression de RAR.

Les archives de format 7z, XZ et LZMA ZIP sont décompressées à l'aide du LZMA SDK (domaine public) d'Igor Pavlov, développeur du logiciel de compression 7-Zip.

Nous avons utilisé l'article "Screaming Fast Galois Field Arithmetic Using Intel SIMD Instructions" de James S. Plank, Kevin M. Greenan et Ethan L. Miller pour améliorer les performances de codage Reed-Solomon. Nous adressons également nos remerciements à Artem Drobanov et Bulat Ziganshin pour les exemples et les idées qui ont permis d'accroître l'efficacité du codage Reed-Solomon.

Les éléments graphiques de RAR ont été conçus par Aditya Nugraha Putra de weirdsgn.com.
