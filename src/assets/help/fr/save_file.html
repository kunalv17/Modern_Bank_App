Enregistrement des fichiers

Tapez sur les noms de dossier pour les ouvrir, vous pouvez ainsi sélectionner un dossier pour enregistrer le fichier. Vous pouvez également modifier le nom du fichier. Appuyez sur "OK" pour enregistrer le fichier dans le dossier ouvert en cours.

Utilisez le Panneau de Navigation pour sélectionner le dossier en cours dans la liste des dossiers favoris. Un appui long sur n'importe quel fichier ou nom de dossier ouvre le menu contextuel, qui permet de créer un nouveau dossier.
