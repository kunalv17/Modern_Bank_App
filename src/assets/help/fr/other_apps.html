Interagir avec d'autres applications

Il est possible d'utiliser les fonctions de RAR à l'intérieur d'un grand nombre d'autres applications de gestion de fichiers. Les approches habituelles appropriées pour de telles applications sont énumérées ci-dessous.

Tapez sur le nom de l'archive ou appliquez la commande "Ouvrir avec" puis sélectionnez RAR dans la liste des applications pour ouvrir une archive dans RAR.

De nombreux gestionnaires de fichiers comprennent une commande nommée "Partager" ou "Envoyer" permettant d'envoyer des fichiers vers d'autres applications. Si vous sélectionnez des fichiers, activez cette fonction et choisissez RAR comme application de destination, vous serez ainsi en mesure de définir les paramètres de compression dans la boîte de dialogue d'archivage et de compresser les fichiers sélectionnés. Si le dossier contenant les fichiers sélectionnés est accessible en écriture, RAR le proposera pour créer une archive, sinon un autre dossier accessible en écriture sera sélectionné.

Si vous avez sélectionné une seule archive et utilisé la commande "Partager" ou "Envoyer", RAR va l'extraire au lieu de l'archiver à nouveau.

Après les commandes "Partager" ou "Envoyer", RAR se ferme automatiquement. Si le dossier en cours dans votre application n'est pas actualisé, il peut être nécessaire de le rafraîchir manuellement pour voir l'archive nouvellement créée ou les fichiers extraits.

Si vous souhaitez envoyer des fichiers de RAR vers une autre application, sélectionnez les fichiers dans la liste de fichiers RAR et utilisez la commande "Envoyer" dans le Menu Options.
