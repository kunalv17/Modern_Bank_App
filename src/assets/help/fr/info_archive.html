Informations sur l'archive

Cette boîte de dialogue fournit des informations détaillées sur l'archive ouverte en cours.

Le champ "Type" inclut le format de l'archive, solide, SFX (auto-extractible) et les attributs du volume. "Version à extraire" contient la version minimale exigée du logiciel de décompression en fonction du format de l'archive qui n'est pas nécessairement liée à la version en cours de RAR. Pour certains formats d'archive , tels que ZIPX, ce champ peut également contenir un nom d'algorithme de compression. "OS" contient le système d'exploitation utilisé pour créer l'archive.

Le paramètre "Dossier de restauration" n'est disponible que pour les archives RAR. Certaines données sont disponibles pour certains formats d'archives, mais pas pour d'autres.

Si un commentaire d'archive est présent, son contenu s'affiche dans la partie inférieure de la boîte de dialogue. Le contenu du commentaire est déroulant. Le bouton "Modifier" est affiché si un commentaire est présent, et "Ajouter" s'il est manquant. Ces boutons sont accessibles pour les archives qui peuvent être modifiées par RAR. Une nouvelle fenêtre s'affiche pour saisir un commentaire ou le charger à partir du fichier. Utilisez le format UTF-8 pour le fichier contenant le commentaire à charger.
