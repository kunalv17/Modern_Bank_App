Chercher un fichier (archive)

Cette boîte de dialogue peut afficher un nom de fichier complet, y compris son chemin d'accès, mettre à jour le chemin d'accès au fichier en gardant son nom ou indiquer le nom d'un dossier.

Tapez sur les noms des dossiers pour les ouvrir, afin de trouver le dossier contenant les fichiers requis. Si vous trouvez le fichier ou l'archive dont vous avez besoin, tapez sur son nom.

Si vous n'avez besoin que du nom d'un dossier, mais pas d'un nom de fichier, appuyez sur le bouton "Dossier seulement". Son comportement peut varier en fonction des différents modules de RAR. Lorsqu'il est invoqué depuis la boîte de dialogue d'archivage, il met à jour l'intitulé du chemin d'accès au nom de l'archive en gardant les composants du nom d'origine. A partir d'autres modules de RAR il peut afficher le nom complet du dossier sans conserver le nom d'origine.

Utilisez le Panneau de Navigation pour sélectionner le dossier en cours dans la liste des dossiers favoris. Un appui long sur n'importe quel fichier ou nom de dossier ouvre le menu contextuel, qui permet de créer un nouveau dossier.
