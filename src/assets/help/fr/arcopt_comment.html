Options d'archivage: Commentaire

Charger le commentaire depuis un fichier

Indiquez un chemin de fichier pour ajouter son contenu en tant que commentaire d'archive. Ce fichier doit être au format texte UTF-8. Utilisez le bouton "Parcourir" pour rechercher un fichier.

Saisir un commentaire manuellement

Saisissez le texte à ajouter comme commentaire d'archive. Les commentaires multilignes sont autorisés.

Vous pouvez soit charger un commentaire depuis fichier ou en saisir un manuellement. L'utilisation simultanée des deux options n'est pas possible.
