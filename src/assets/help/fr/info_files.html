Information

Cette boîte de dialogue s'affiche pour les fichiers et dossiers courants sélectionnés en dehors des archives. Il fournit des informations sur le nombre total de fichiers et dossiers et la taille totale des éléments sélectionnés y compris les fichiers dans les sous-dossiers.

Si un seul élément est sélectionné, son nom complet, y compris son chemin d'accès, s'affiche en haut de la boîte de dialogue. Vous pouvez faire un clic long dessus et utiliser "Copier" pour le placer au format texte dans le presse-papiers.

Vous pouvez également y calculer les sommes de contrôles MD5, SHA-1, SHA-256 et CRC32 pour tous les fichiers sélectionnés y compris les fichiers dans des dossiers sélectionnés. Cela peut s'avérer utile pour vérifier l'intégrité des téléchargements à condition que la somme de contrôle correcte du fichier soit connue. Appuyez sur le bouton approprié pour démarrer le calcul. Lorsque le calcul est fait, vous pouvez appuyer sur bouton «Enregistrer» pour enregistrer toutes les sommes de contrôle et les noms de fichiers dans un fichier ou "Copier" pour les placer dans le presse-papiers du système, afin de pouvoir les coller dans une autre application. En tapant sur un élément dans la liste des résultats, cette somme de contrôle est copiée dans le presse-papiers.

La somme de contrôle CRC32 utilise l'algorithme compatible avec CRC32 dans les archives RAR et ZIP.

Utilisez le bouton "Copier les noms" pour placer les noms des fichiers sélectionnés dans le presse-papiers du système. Cela permet de les coller sous forme de texte dans d'autres applications.
