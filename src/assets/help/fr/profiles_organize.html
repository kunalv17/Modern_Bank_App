Organiser les Profils

Appuyez sur les cases à cocher ou les noms de profils pour sélectionner un élément. Vous pouvez sélectionner un ou plusieurs profils. Lorsque toutes les modifications sont effectuées, appuyez sur le bouton OK pour les enregistrer.

Les commandes suivantes sont disponibles:

Tout sélectionner

Sélectionnez tous les profils de compression dans la liste. Si tous les profils sont déjà sélectionnés, cette commande les désélectionne.

Effacer

Supprimez les profils sélectionnés.

Définir par défaut

Attribuer le statut 'par défaut' au profil sélectionné. Ce profil sera utilisé pour définir les paramètres initiaux dans la boîte de dialogue d'archivage. Si plusieurs profils sont sélectionnés, le premier d'entre eux obtient le statut par défaut.
