Confirmez le fichier à remplacer

Cette invite s'affiche lorsque les fichiers extraits ou copiés ont le même nom que des fichiers existants.

Sélectionnez "Suivant" pour sauter au fichier suivant et le préserver en l'état. Sélectionnez "Remplacer" pour remplacer le fichier existant par le nouveau. "Ignorer tout" et "Remplacer tout" répéteront les actions "Suivant" ou "Remplacer" aussi bien pour les conflits de nom de fichier en cours que les suivants.

Si vous souhaitez renommer le nouveau fichier manuellement, modifier le nom du fichier directement dans la boîte de dialogue. Le champ contenant le nom du fichier à créer est modifiable. Lorsque vous entrez un nom de votre choix, appuyez sur le bouton "Renommer" ou "Remplacer". "Renommer" et "Remplacer" ont le même effet à la seule différence que "Renommer" ne permet pas de continuer si les noms d'origine et les nouveaux sont identiques. Si le nom de fichier nouvellement spécifié existe déjà, la boîte de dialogue de confirmation de changement de nom s'affiche à nouveau.

Si vous avez appuyé sur "Renommer automatiquement", tous les fichiers créés qui sont en conflit avec des fichiers existants obtiendront des noms comme «nom de fichier (ET) .ext», où «filename.ext 'est le nom du fichier original et' N 'est un nombre.

Selon le type de commande, les options de remplacement de nom peuvent ne pas être disponibles.

Appuyez sur "Annuler" pour mettre fin à l'ensemble de l'opération.
