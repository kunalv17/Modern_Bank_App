Les options d'extraction

Si la commande d'extraction est invoquée à l'intérieur d'une archive, tout en parcourant le contenu des archives, cela décompressera les fichiers et les dossiers sélectionnés dans cette archive. Si rien n'est sélectionné, RAR sélectionne automatiquement le contenu entier de l'archive.

Si l'extraction est effectuée à l'extérieur de l'archive, tout en naviguant dans les fichiers courants non archivés, les fichiers sélectionnés sont considérés comme des archives et traités pendant l'extraction. Si des dossiers sont sélectionnés, RAR va rechercher les fichiers avec des extensions d'archives tels que .rar, .zip, etc., et tenter de les extraire également. Si rien n'est sélectionné, RAR sélectionne automatiquement dans le dossier en cours tous les fichiers avec des extensions d'archives.

Les options disponibles dans cette boîte de dialogue sont énumérées ci-dessous.

Chemin de destination

Désigne le nom du dossier pour placer les fichiers extraits. Si ce dossier n'existe pas, il sera créé automatiquement. Vous pouvez accepter l'opération de création du nom de dossier par défaut, entrez un autre nom de dossier à l'aide d'un clavier ou appuyez sur le bouton "Parcourir" pour naviguer vers un dossier de destination. Vous pouvez également cliquer sur le bouton "Dossier de destination" pour sélectionner rapidement l'une des options proposées dans la liste. Les détails sur la liste des chemins proposés sont fournis ici.

Confirmer avant d'effacer

Confirmez avant d'effacer un fichier existant.

Effacer sans confirmer

Les fichiers existants sont remplacés sans aucune confirmation.

Ignorer les fichiers existants

Ne pas effacer les fichiers qui existent déjà.

Conserver les fichiers endommagés

Si activé, RAR ne supprimera pas les fichiers qui ne se sont pas extraits correctement, par exemple, lorsque l'archive est endommagée. Par défaut, RAR supprime ces fichiers. Vous pouvez essayer de sauver une partie des données d'un fichier endommagé.

Ne pas extraire les chemins

Ignorer les données relatives au chemins d'accès de l'archive et extraire les fichiers du dossier de l'archive racine et ses sous-dossiers dans le même dossier de destination.

Extraire des archives dans les sous-dossiers

Cette option est disponible uniquement lorsque plusieurs archives sont extraites et sans effet pour une archive unique. Le contenu non extrait de chaque archive est placé dans un sous-dossier distinct, dont le nom sera généré en se basant sur le nom de l'archive.

Afficher les fichiers extraits

Affiche le contenu du dossier avec les fichiers extraits après la réussite de l'opération.

Enregistrer les paramètres

Définir l'état actuel des options de cette boîte de dialogue en tant que nouvelle valeur par défaut. Toutes les options excepté le chemin de destination sont enregistrées. "Extraire les archives dans les sous-dossiers " n'est enregistré que si vous avez sélectionné plusieurs archives. Les options enregistrées sont également utilisées dans le menu contextuel et les opérations d'extraction du presse-papiers.
