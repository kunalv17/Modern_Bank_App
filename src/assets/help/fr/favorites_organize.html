Organiser les favoris

Appuyez sur les cases à cocher ou les noms préférés pour sélectionner un élément. Vous pouvez sélectionner un ou plusieurs favoris. Lorsque toutes les modifications sont effectuées, appuyez sur le bouton OK pour les enregistrer.

Les commandes suivantes sont disponibles:

Tout sélectionner

Sélectionnez tous les favoris dans la liste. Si tous les favoris sont déjà sélectionnés, cette commande les désélectionne.

Effacer

Supprimez les favoris sélectionnés.

Editer

Modifiez les paramètres d'un favori sélectionné. Si plusieurs favoris sont sélectionnés, seul le premier d'entre eux sera changé.
