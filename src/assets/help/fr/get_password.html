Ajouter un mot de passe

Indiquez un mot de passe pour crypter les fichiers lors de l'archivage ou pour décrypter un fichier verrouillé lors de l'extraction.

Pour une sécurité maximale quand vous archivez des fichiers avec un mot de passe il est recommandé:

  • d'utiliser le format RAR, car il utilise un algorithme de chiffrement plus fort;
  • d'éviter les mots de passe de moins de 8 caractères;
  • d'éviter d'utiliser comme mots de passe les mots les plus courants.

N'oubliez pas vos mots de passe. Le chiffrement de RAR ne contient pas d'accès de secours. Les développeurs de RAR ne sont pas en mesure de vous aider à restaurer vos mots de passe perdus.

Montrer le mot de passe

Masquez ou affichez le mot de passe pendant la saisie.

Crypter les noms de fichier

Cette option n'est disponible que lors de la création d'archives au format RAR. Par défaut, si cette option est désactivée, seules les données des fichiers sont cryptées. Les noms de fichiers, les tailles et les dates sont encore visibles sans mot de passe. Lorsque vous activez cette option, aussi bien les données des fichiers que leurs noms, tailles, dates sont cryptés. Le mot de passe est nécessaire, même pour voir la liste des fichiers contenus dans l'archive.

Se souvenir pour <N> minutes

Cette option n'est disponible que lors de l'extraction, du test ou de la navigation dans les archives. Elle permet de se souvenir d'un mot de passe saisi et de l'appliquer à toutes les commandes d'extraction pendant le nombre de minutes spécifié. Cela peut s'avérer pratique lors de l'extraction de nombreuses archives avec un même mot de passe. Le mot de passe est utilisé jusqu'à l'expiration du temps spécifié, d'une erreur de données trouvée ou si RAR est fermé avec le bouton Retour.
