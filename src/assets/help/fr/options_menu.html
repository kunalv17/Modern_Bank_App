Menu "Options"

Ce menu s'affiche lorsque vous appuyez sur l'icône dans le coin en haut à droite de RAR ou avec le bouton menu de l'appareil, si disponible.

Fichiers de test des archives

Vérifiez les données archivées et signalez les erreurs éventuellement trouvées.

Réparation d'une archive

Essayez de réparer l'archive sélectionnée ou ouverte.

Comparer

Comparer les performances du dispositif.

Envoi

Envoyez les fichiers sélectionnés vers d'autres applications. Peut s'utiliser à la fois pour les fichiers archivés et les fichiers courants. Si activé à l'intérieur de l'archive et qu'aucun fichier n'est sélectionné, l'ensemble de l'archive est envoyé. Si certains fichiers archivés sont sélectionnés, ils sont décompressés dans le dossier temporaire, puis envoyés.

Information

Affiche des informations sur les fichiers et dossiers sélectionnés si elle est invoquée en dehors de l'archive. Affiche des informations sur l'archive si elle est invoquée à l'intérieur de l'archive.

Retirer les publicités

Vous pouvez payer pour enlever les publicités dans RAR. Après avoir effectué l'achat, cet élément du menu devient indisponible. Il est également indisponible lorsque la version actuelle de RAR n'inclut pas de publicités.

Paramètres

Ouvre la boîte de dialogue Paramètres pour configurer l'interface de RAR et d'autres paramètres.

Aide

Affiche le contenu de l'aide

Quitter

Fermer RAR.
