Support de carte SD externe

Ce sujet concerne uniquement les cartes externes SD amovibles dans les appareils Android 4.4 uniquement. Les autres version d'Android ne limitent pas l'enregistrement sur la carte externe. Les cartes internes sont entièrement accessibles en écriture, quelle que soit la version d'Android.

A partir d'Android 4.4, seules les applications pré-installées dans le matériel du fabricant peuvent écrire vers les dossiers d'une carte SD externe. Les autres applications, y compris RAR, ne sont seulement autorisées qu'à écrire dans un dossier spécifique sur la carte externe et sont limitées en lecture seule pour les autres dossiers des cartes externes. Pour RAR, le dossier de la carte externe accessible en écriture est "Android / data / com.rarlab.rar". RAR permet la création d'un sous-dossier "fichiers" pour stocker les archives et les fichiers et utilise le nom "Accueil de la carte SD externe" pour ce dossier. En supposant qu'une carte SD externe est disponible dans /storage/extSdCard (cela varie en fonction des appareils), le nom complet de l'accueil de la carte SD externe sera /storage/extSdCard/Android/data/com.rarlab.rar/files.

Pour les appareils avec carte SD externe et sous Android 4.4, les liens vers le dossier racine de la carte SD (en lecture seule) et l'accueil de la carte SD externe (accessible en écriture) sont ajoutés aux "Favoris" dans le panneau de navigation après nouvelle installation de RAR. Vous pouvez également forcer RAR à créer ces lien en supprimant tous les articles préférés dans le panneau de navigation.

Veuillez bien noter que lorsque vous désinstallez RAR ou des données de RAR dans le gestionnaire d'applications Android, le contenu de son dossier d'accueil dans la carte SD externe et ses sous-dossiers est entièrement effacé. Copiez tous les fichiers importants de ce dossier à un autre endroit avant d'effectuer une telle opération.
