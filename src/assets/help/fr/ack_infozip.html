Code source Info-ZIP

RAR intègre le code source du groupe Info-ZIP, utilisé pour manipuler les archives ZIP. La source d'origine Info-ZIP est disponible gratuitement depuis ftp://ftp.info-zip.org/pub/infozip/ et www.info-zip.org.
