Options d'archivage: Paramètres Généraux

Nom de l'archive

Indiquez le nom de l'archive pour créer ou mettre à jour. Vous pouvez entrer le nom manuellement ou appuyez sur bouton "Parcourir" pour localiser une archive ou un dossier existant. Si l'extension de nom de l'archive n'est pas spécifiée, elle sera ajoutée automatiquement. Si le chemin d'accès à l'archive est manquant, le dossier en cours est pris en compte. Le libellé « Nom de l'archive » est cliquable et permet de basculer entre le dossier actuel et le dossier racine de la mémoire du périphérique. Dans Android 4.4 ou plus récent, « nom de l'archive » affiche également les chemins de la carte externe et du lecteur USB quand ils sont disponibles.

RAR, ZIP et RAR 4.x

Sélectionnez le format d'archive. L'option RAR signifie le format d'archive RAR 5.0, qui fournit habituellement de meilleurs taux de compression, de restauration des données et d'options de chiffrement, alors que le format ZIP est plus répandu et peut être décompressé par un plus grand nombre d'autres logiciels. Sélectionnez RAR 4.x pour créer une archive au format 4.x RAR, compatible avec les versions 4.x de WinRAR. Le format RAR 5.0 permet un enregistrements de restauration plus efficace, un cryptage plus fort et généralement un taux de compression plus élevé que RAR 4.x. Il ne faut donc utiliser RAR 4.x que si l'archive doit être décompressée par un logiciel ne prenant pas en charge le format de décompression RAR 5.0.

Définir un mot de passe

Protégez l'accès à vos archives de fichiers à l'aide d'un mot de passe. Pour une sécurité maximale il est recommandé d'utiliser le format RAR, qui utilise un algorithme de cryptage plus fort, d'éviter les mots de passe plus courts que 8 caractères et les mots de passe trop simples.

Supprimer les fichiers après archivage

Si l'archivage a réussi, les fichiers d'origine seront supprimés.

Création d'archives solides

Cette option est disponible uniquement pour le format RAR et seulement pour les nouvelles archives. Il ne fonctionne pas sur la mise à jour d'une archive existante. Si elle est activée, tous les fichiers dans l'archive sont traités comme un flux de données continu. Cela contribue à améliorer le taux de compression, en particulier pour les multiples petits fichiers similaires. Bien que la création d'une nouvelle archive solide nécessite environ le même temps qu'une archive non-solide courante, la mise à jour des fichiers et l'extraction de fichiers individuels d'archives solides est beaucoup plus lente que dans les archives non-solides. Il n'est pas recommandé d'utiliser le mode solide si vous prévoyez de mettre à jour une archive à l'avenir. Les archives solides sont également plus sensibles aux dommages de données. Contrairement aux archives habituelles, tous les fichiers seront très probablement également endommagées si une archive est endommagée

Profils

Ce bouton permet d'enregistrer les réglages actuels de toutes les boîtes de dialogue d'archivage dans un profil, de sélectionner les paramètres précédemment enregistrés dans les profils et d'organiser les profils. Vous pouvez marquer l'un des profils comme profil par défaut dans "Organiser les profils", ainsi il sera sélectionné automatiquement lors de l'ouverture de la boîte de dialogue. Le Profil par défaut est désigné par un cercle plein devant son nom, les autres profils ont un cercle vide qui précède leur nom dans la liste des profils enregistrés. La commande "Enregistrer les paramètres actuels par défaut" enregistre en une seule étape les paramètres en cours en paramètres par défaut en se servant du nom du "profil par défaut". Certains paramètres, comme le nom de l'archive, le mot de passe et les fichiers sélectionnés, ne sont pas enregistrés dans le profil. Une partie des options du profil par défaut est également utilisée dans le menu contextuel et dans les opérations d'archivage du presse-papiers.
