Introdução à interface

A janela principal do RAR contém a lista de arquivos e pastas na pasta ou arquivos comprimidos atualmente abertos. O item da parte superior da lista exibe tanto como espaço livre e tamanho total para dispositivos de armazenamento ou tamanhos de compressão e descompressão de arquivos comprimidos. Acima desta lista existem botões que chamam os principais comandos do RAR, título do RAR, opções do botão de menu de acesso para comandos adicionais. Barra de status abaixo da lista de arquivos exibe a pasta ou nome do arquivo atual ou informações sobre os arquivos selecionados.

Pressione qualquer pasta ou nome do arquivo na lista de arquivos para procurar pastas ou conteúdo do arquivo. A mesma ação para arquivos habituais abre o arquivo no programa associado. Pressionar a parte superior do item da lista para ir à pasta principal ou fechar um arquivo em caso de pasta raiz de arquivos comprimidos.

Toque na caixa de seleção ao lado direito de qualquer item da lista para selecionar o item. Toque na caixa de seleção superior do item da lista para marcar ou desmarcar todos os itens da lista. Pressione e segure qualquer caixa de seleção para acessar a seleção de opções avançadas.

Use os botões acima da lista e os comandos no menu de opções para o processo de itens selecionados. Use o botão "Adicionar" para adicionar arquivos e pastas selecionados para um novo ou arquivos comprimidos existentes, "Extrair" para extrair arquivos comprimidos selecionados, "Apagar" para apagar arquivos e pastas. Se você tiver selecionado um nome de arquivo comprimido, "Extrair" vai descomprimir todo o arquivo. Você também pode abrir o conteúdo dos arquivos e selecionar arquivos individuais para descomprimir. É possível selecionar vários arquivos e descomprimi-los todos em uma única operação.

Se nenhum arquivo for selecionado, "Adicionar" comando seleciona automaticamente todos arquivos e pastas na pasta atual. No mesmo caso "Extrair" e "Testar" seleciona todos os arquivos comprimidos com base em extensões de arquivo ou seleciona todo o conteúdo do arquivo comprimido, se estiver dentro do arquivo comprimido.

Pressionando e segurando qualquer item na lista de arquivo é aberto o menu de contexto. Deslize da borda esquerda da tela ou clique no ícone do RAR ou título no canto superior esquerdo para trazer o painel de navegação.

Pressione e segure o botão Voltar na janela principal para fechar o RAR imediatamente em vez de ir para a pasta anterior.
