Pasta padrão de extração

Especifique uma pasta que será proposta por padrão como destino para arquivos extraídos no diálogo de extração e menu de contexto.

Pasta padrão do usuário

Define quando usar uma pasta padrão de extração. As opções possíveis são "Nunca", "Apenas dentro da pasta temporária" e "Sempre". Todo o conteúdo da pasta temporária Android/data/com.rarlab.rar é removido ao desinstalar o aplicativo RAR e pode ocorrer perda de dados se houver arquivos descomprimidos anteriormente. Então, por padrão, esta opção é definida como "Apenas dentro da pasta temporária" e se o arquivo é armazenado na pasta temporária, que pode acontecer se ele foi recebido de outro aplicativo, o RAR automaticamente propõe para descomprimir esse arquivo para a pasta padrão de extração, fora pasta temporária.

Anexar nome do arquivo

Controla se um nome do arquivo sem extensão é anexado ao caminho de destino proposto inicialmente no diálogo de extração. Ela não afeta a extração para a pasta de comando no menu de contexto.

Histórico de extração

Se ativada, caminhos de destino no diálogo de extração são adicionados ao histórico e podem ser selecionados posteriormente, com o botão "Caminho do destino" neste diálogo. Caminhos selecionados diretamente do menu de contexto não são adicionados ao histórico independentemente desta opção.

Pasta de inicialização

Seleciona uma pasta para ser exibida depois do RAR ser inicializado. Por padrão é a pasta raiz do dispositivo de armazenamento.
