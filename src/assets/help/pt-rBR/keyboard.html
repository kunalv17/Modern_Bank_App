Teclado físico e mouse

O RAR suporta os seguintes atalhos adicionais para teclado físico conectado ao dispositivo Android:

Alt+A
Adicionar à compressão
Alt+E
Extrair
Ctrl+A
Selecionar todos os arquivos
Ctrl+C
Copiar
Ctrl+V
Colar
Ctrl+X
Recortar
Del
Excluir arquivos
Ins
Inverter seleção de arquivos
Shift+Up, Shift+Down
Selecionar arquivos

Se um mouse físico estiver conectado e o dispositivo suportar cliques, clique com o botão direito em um arquivo para exibir o menu de contexto.
