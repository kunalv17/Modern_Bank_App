Menu Opções

Este menu é exibido ao pressionar o ícone no canto direito superior do RAR ou com o botão Menu do dispositivo, se disponível.

Testar arquivos comprimidos

Verifica os dados arquivados e reporta erros, se forem encontrados.

Reparar arquivo

Tenta reparar o arquivo aberto ou selecionado.

Desempenho

Desempenho do dispositivo.

Enviar

Envia os arquivos selecionados para outros aplicativos. Pode ser usado tanto para arquivos comprimidos e habituais. Se ativado dentro dos arquivos comprimidos e nenhum arquivo for selecionado, todo o arquivo é enviado. Se alguns arquivos comprimidos forem selecionados, eles serão descomprimidos para a pasta temporária e, em seguida, enviados.

Informações

Exibe informações sobre arquivos e pastas selecionadas se chamado de fora de um arquivo comprimido. Exibe informações de compressão se chamado de dentro de um arquivo comprimido.

Remover anúncios

Você pode pagar para remover os anúncios do RAR. Depois que a compra é concluída, este item de menu torna-se indisponível. Também não estará disponível se a versão RAR atual não incluir anúncios.

Ajustes

Abre o diálogo Ajustes para configurar a interface do RAR e outros parâmetros.

Ajuda

Exibe o conteúdo da ajuda.

Sair

Fecha o RAR.
