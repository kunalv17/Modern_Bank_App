Organizar perfis

Toque na caixa de verificação ou nomes de perfil para selecionar um perfil. Você pode selecionar um ou vários perfis. Quando todas as modificações forem concluídas, pressione o botão OK para armazená-los.

Os seguintes comandos estão disponíveis:

Selecionar tudo

Seleciona todos os perfis de compressão na lista. Se todos os perfis já estão selecionados, este comando vai desmarcá-los.

Apagar

Apaga os perfis selecionados.

Definir como padrão

Atribui status de 'padrão' para o perfil selecionado. Esse perfil será usado para definir os parâmetros iniciais no diálogo de compressão. Se vários perfis são selecionados, o primeiro deles obtém o status padrão.
