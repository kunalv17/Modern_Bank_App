Tema

Selecione o tema da interface escuro, claro ou preto.

Som

Emitir um som após concluir uma operação. A notificação sonora não é emitida se a operação for concluída rapidamente.

Ocultar barra de notificação

Oculta a barra de notificação do sistema na parte superior da tela.

Tamanho da fonte

Altera o tamanho da fonte para a lista de arquivos, a barra de status e algumas outras partes da interface. Escolha valores positivos para fonte maior e negativo para fonte menor.
