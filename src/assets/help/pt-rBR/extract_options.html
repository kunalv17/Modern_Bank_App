Opções de extração

Se o comando de extração é chamado de dentro do arquivo, enquanto navega o conteúdo do arquivo, ele irá descomprimir os arquivos e pastas selecionados neste arquivo. Se nada estiver selecionado, o RAR selecionará automaticamente todo o conteúdo do arquivo.

Se a extração é chamada de fora do arquivo comprimido, enquanto navega habitualmente nos arquivos não comprimidos, os arquivos selecionados são considerados arquivos comprimidos e processados ​​com a descompressão. Se as pastas são selecionadas, o RAR irá procurá-los de forma recursiva em arquivos com extensões de arquivo como .rar, .zip, etc, e vai tentar descomprimi-los também. Se nada for selecionado, o RAR irá selecionar automaticamente todos os arquivos com as extensões do arquivo na pasta atual.

As opções disponíveis neste diálogo estão listadas abaixo.

Caminho de destino

Nome da pasta para colocar os arquivos extraídos. Se esta pasta não existir, ela será criada automaticamente. Você pode aceitar o nome de pasta proposto por padrão, ou digitar outro nome de pasta usando um teclado ou pressionando o botão "Procurar" para procurar a pasta de destino. Além disso, Além disso, você pode clicar em "Caminho de destino" para selecionar rapidamente um dos caminhos propostos da lista. Detalhes sobre a lista de caminhos propostos são fornecidos aqui.

Perguntar antes de substituir

Perguntar antes de substituir um arquivo existente.

Substituir sem perguntar

Os arquivos existentes são substituídos sem qualquer aviso.

Ignorar arquivos existentes

Não substituir arquivos que já existem.

Manter arquivos corrompidos

Se ativado, o RAR não vai apagar os arquivos que não são extraídos corretamente, por exemplo, quando o arquivo estiver danificado. Por padrão, o RAR apaga esses arquivos. Você pode tentar salvar uma parte da informação do arquivo corrompido.

Não extrair caminhos

Ignora as informações do caminho arquivado e extrai arquivos da pasta e subpastas raiz do arquivo comprimido para a mesma pasta de destino.

Extrair arquivos para subpastas

Esta opção está disponível apenas quando vários arquivos comprimidos são descomprimidos e desativada para um único arquivo. Ele coloca o conteúdo descomprimido de cada arquivo em uma subpasta separada, cujo nome será gerado com base no nome do arquivo.

Exibir arquivos extraídos

Exibe o conteúdo da pasta com os arquivos extraídos ao concluir a extração.

Salvar ajustes

Define o estado atual das opções nesta caixa de diálogo como o novo padrão. Todas as opções, exceto o caminho de destino são salvos. "Extrair arquivos para subpastas" é salvo apenas se você selecionou vários arquivos. As opções salvas aqui também são usadas nas operações de extração no menu de contexto e área de transferência.
