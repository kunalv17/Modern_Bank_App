Formatos de Compressão

O compressor RAR para Android suporta os seguintes formatos de compressão: RAR, ZIP, 7-Zip (.7z), ARJ, BZip2 (.bz2), GZip (.gz), ISO9660 (.iso), Lzip (.lz), LZH, TAR, UDF (.iso), XZ, compressor Unix (.Z). A compressão e extração é possível para formatos RAR e ZIP, extração apenas para o restante dos formatos.

Arquivos 7-Zip são descomprimidos com o SDK LZMA, por isso apenas subconjunto de arquivos .7z suportados pelo SDK LZMA podem ser descomprimidos pelo compressor RAR.

Arquivos compostos .tar.gz, tar.bz2, tar.lz, tar.xz, tar.Z são processados em um único passo, assim o usuário vê os arquivos no TAR depois de abrir ou descomprimir esse arquivo.
