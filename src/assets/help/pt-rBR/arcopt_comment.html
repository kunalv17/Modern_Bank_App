Opções de Compressão: Comentários

Carregar um comentário do arquivo

Especifique um caminho de arquivo para adicionar seu conteúdo como comentário de arquivo. Esse arquivo deve ter o formato de texto UTF-8. Use o botão "Procurar" para localizar um arquivo.

Inserir um comentário manualmente

Digite o texto para ser adicionado como comentário do arquivo. São permitidos comentários com várias linhas.

Você pode carregar comentários de um arquivo ou inseri-lo manualmente. Não é possível usar as duas opções juntas.
