Organizar favoritos

Toque na caixa de verificação ou nomes favoritos para selecionar o item. Você pode selecionar um ou vários favoritos. Quando todas as modificações forem feitas, pressione o botão OK para armazená-las.

Os seguintes comandos estão disponíveis:

Selecionar tudo

Selecione todos os favoritos na lista. Se todos os favoritos já estiverem selecionados, este comando vão desmarcá-los.

Apagar

Apaga os favoritos selecionados.

Editar

Modifica os parâmetros de favorito selecionado. Se diversos favoritos são selecionados, apenas o primeiro deles vai ser editado.
