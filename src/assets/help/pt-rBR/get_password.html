Inserir senha

Especifica uma senha para criptografar arquivos ao comprimir ou descomprimir um arquivo criptografado ao extrair.

Se você comprimir arquivos com senha, para uma máxima segurança, recomenda-se:

  • usar o formato RAR, porque utiliza um algoritmo de criptografia mais forte;
  • evitar senhas mais curtas que 8 caracteres;
  • evitar palavras comuns como senha.

Lembre-se das suas senhas. A criptografia do RAR não possibilita recuperação. Os desenvolvedores do RAR não são capazes de ajudá-lo a restaurar suas senhas.

Mostrar senha

Oculta ou mostra a senha ao você digitar.

Criptografar nomes de arquivo

Esta opção só está disponível na criação de arquivos comprimidos de formato RAR. Por padrão, se esta opção estiver desativada, apenas os dados do arquivo são criptografados. Os nomes de arquivo, tamanhos, datas ainda são visíveis sem senha. Ao ativar esta opção, ambos os dados de arquivos e nomes de arquivos, tamanhos, datas são criptografados, então a senha é necessária mesmo para ver a lista de arquivos contidos no arquivo.

Lembrar em <N> minutos

Esta opção está disponível apenas quando ao extrair, testar ou navegar arquivos. Ela permite lembrar uma senha digitada e aplicá-la a todos os comandos de extração durante determinado tempo em minutos. Pode ser conveniente quando você precisar descomprimir vários arquivos com a mesma senha. A senha é usada até o tempo de expiração especificado ou se erro de dados forem encontrados ou se o RAR for fechado com o botão Voltar.
