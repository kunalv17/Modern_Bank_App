Tastiera e mouse reali

RAR supporta le seguenti scorciatoie per le tastiere fisiche connesse ad un dispositivo Android:

Alt+A
Aggiungi ad un archivio
Alt+E
Estrai da un archivio
Ctrl+A
Seleziona tutti i file e cartelle(All)
Ctrl+C
Copia file o cartella
Ctrl+V
Incolla file o cartella
Ctrl+X
Taglia file o cartella
Del
Elimina file o cartella
Ins
Inverti l'attuale selezione dei file/cartelle
Maius+Su, Maius+Giù
Seleziona file/cartella

Se viene connesso un mouse ed il dispositivo supporta la gestione del tasto destro, cliccando con il tasto destro del mouse su di un file o cartella verrà visualizzato il relativo menu contestuale.
