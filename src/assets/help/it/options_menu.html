Menu opzioni

Questo menu viene visualizzato quanto selezionate l'icona in alto a destra (dei 3 riquadri sovrapposti) o quando selezionate il pulsante "Menu", se presente sul vostro dispositivo.

Verifica file archiviati

Esegue una verifica dell'integrità dei dati archiviati e se individua degli errori mostra un rapporto dettagliato.

Ripara archivio

Prova a riparare o l'archivio selezionato o quello aperto in questo momento.

Valutazione sistema

Valuta le prestazioni del vostro dispositivo Android utilizzando l'algoritmo RAR.

Condividi

Selezionando questa voce potete condividere i file selezionati con altre applicazioni e da queste con altri utenti. Può essere utilizzato sia per file archiviati che per file normali.
Se avete aperto un archivio e non avete selezionato alcune file, verrà condiviso l'intero archivio; se invece è selezionato qualche file dell'archivio, questi verranno estratti in un cartella temporanea e quindi condivisi.

Informazioni

Visualizza delle informazioni sui file e/o cartelle selezionate se invocata da fuori un archivio. Visualizza le informazioni dell'archivio se questa funzione viene invocata dall'interno di un archivio.

Rimuovi la pubblicità

Potete passare alla versione di RAR senza pubblicità pagando una piccola somma. Una volta eseguito l'acquisto, oltre a scomparire la pubblicità, non verrà più mostrata neanche questa voce del menu. Questa voce può non essere presente se la versione corrente di RAR non prevede la pubblicità.

Configurazione

Apre il Pannello di configurazione per modificare la configurazione dell'interfaccia di RAR e di altri parametri

Guida

Visualizza il menu principale della Guida.

Esci

Chiude RAR.
