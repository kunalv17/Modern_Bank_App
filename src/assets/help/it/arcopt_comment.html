Opzioni d'archiviazione: Commento

Carica il commento da un file

Si può specificare il percorso di un file di testo il cui contenuto diventerà il commento dell'archivio. Questo file deve avere il formato di testo UTF-8. Tocca sul pulsante "Sfoglia..." per localizzare il file sul dispositivo.

Inserisci un commento manualmente

Inserisci manualmente il testo del commento dell'archivio. È permesso inserire un commento su più linee.

Potete o caricare il commento da un file o inserirlo manualmente: non è possibile utilizzare entrambe le opzioni.
