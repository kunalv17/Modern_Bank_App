Inserisci la parola chiave

Specifica una parola chiave per cifrare i file quando verranno archiviati o per decifrarli quando verranno estratti.

Se cifrate i file con una parola chiave, per una sicurezza maggiore si raccomanda di:

  • utilizzare il formato RAR che utilizza un algoritmo di cifratura più sicuro;
  • evitate parole chiavi inferiori agli 8 caratteri;
  • evitate d'utilizzare parole comuni o note come parola chiave.

Ricordatevi la vostra parola chiave: la cifratura di RAR non prevede alcuna scappatoia (backdoor). Nemmeno gli sviluppatori di RAR non riescono ad aprire un archivio cifrato senza essere in possesso della parola chiave.

Cifra i nomi dei file

Questa opzione è disponibile solo se state creando un archivio in formato RAR. Per definizione, se questa opzione non è abilitata, verrà cifrato solo il contenuto dei file mentre i nomi dei file, le dimensioni e le date saranno visibili in chiaro senza aver bisogno della parola chiave.
Se abilitate questa opzione verranno cifrati anche i nomi, le dimensioni e le date dei file archiviati: per questo motivo la parola chiave verrà richiesta anche per visualizzare l'elenco dell'archivio.

Ricorda per <N> minuti

Questa opzione è disponibile solo per operazioni di estrazione, verifica e navigazione di un archivio. Permette di memorizzare la parola chiave inserita ed applicarla a tutti i comandi d'estrazione per la durata, in minuti, qui specificata. Può essere utile quando scompattate più archivi cifrati con la stessa parola chiave. La parola chiave verrà tenuta in memoria sino allo scadere del tempo, o per un errore individuato durante l'apertura di un archivio o se RAR viene chiuso con il pulsante "Indietro".
