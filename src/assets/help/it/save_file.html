Salva file

Selezionate i nomi delle cartelle per aprirle sino ad individuare quella dove memorizzare il file. Potete inoltre modificare il nome del file. Selezionate "Ok" per salvare il file nella cartella attualmente aperta.

Utilizzate il pannello di navigazione per selezionare la cartella di destinazione tra quelle salvate nei Preferiti. Tenete premuto a lungo su qualsiasi nome di file o cartella per aprire il menu contestuale dal quale potete creare una nuova cartella.
