Interazioni con altre applicazioni

È possibile utilizzare le funzioni di RAR all'interno di altre applicazioni di gestione file. Di seguito vengono elencate le funzioni disponibili per queste applicazioni.

Selezionando il nome di un archivio, o selezionando il comando "Apri con" seguito dall'applicazione RAR, vi permettera di aprire l'archivio con RAR per Android.

Inoltre molti gestori di file includono un comando chiamato "Condividi" o "Invia" che permette di passare uno o più file ad un altra applicazione. Se attivate questa funzione partendo dalla selezione di uno o più file e selezionate RAR come applicazione di destinazione, dapprima sarete in grado di impostare i parametri di compressione nel pannello d'archiviazione e di seguito comprimere i file selezionati in un solo archivio. Se la cartella in cui risiedono i file selezionati è "scrivibile" RAR la proporrà come cartella in cui creare l'archivio finale. altrimenti verrà selezionata un'altra cartella.

Se invece selezionate un singolo archivio e utilizzate i comandi "Condividi" o "Invia", RAR avvierà la procedura d'estrazione invece di quella di archiviazione.

Appena processata la funzione "Condividi" o "Invia", RAR verrà chiuso automaticamente. Se il contenuto della cartella corrente nella vostra applicazione non viene aggiornato in automatico, sarà necessario eseguire l'aggiornamento manuale per poter visualizzare o il nuovo archivio creato o i file appena estratti.

Se volete inviare file da RAR ad un'altra applicazione, selezionate i file e utilizzate il comando "Condividi" presente nel menu Opzioni.
