Opzioni d'estrazione

Se il comando d'estrazione viene invocato all'interno di un archivio mentre ne visionate il contenuto, verranno scompattati i file e le cartelle selezionate. Se non viene selezionato alcun oggetto, RAR selezionerà automaticamente l'intero contenuto dell'archivio.

Se il comando d'estrazione viene invocato all'esterno di un archivio, mentre state solitamente scorrendo un elenco di file non-archivi, i file selezionati sono considerati archivi e viene avviato il processo d'estrazione. Se sono selezionate delle cartelle, RAR cercherà al loro interno, anche ricorsivamente, la presenza di file con estensione .RAR, .ZIP, ecc. e avvierà il relativo processo d'estrazione. Se non viene selezionato alcun oggetto RAR selezionerà automaticamente tutti i file con estensione d'archivio presenti nella cartella corrente.

Di seguito sono descritte tutte le opzioni presenti in questo pannello.

Percorso di destinazione

Nome della cartella in cui porre i file estratti. Se questa cartella non esiste verrà creata automaticamente. Potete accettare il nome della cartella proposta per definizione, inserire un altro nome con la tastiera o selezionare il pulsante "Sfoglia" per scorrere le cartelle del dispositivo sino ad individuare la cartella voluta. Potete anche premere sopra il pulsante "Percorso di destinazione" per selezionare velocemente uno dei percorsi proposti nell'apposito elenco. Qui potete leggere ulteriori dettagli in merito alle voci presenti nell'elenco proposto.

Chiedi prima di sovrascrivere

Verrà chiesta la conferma prima di sovrascrivere un file esistente.

Sovrascrivi senza chiedere

Eventuali file già esistenti verranno sovrascritti senza chiederne la conferma.

Salta i file esistenti

Non sovrascrivere i file già esistenti.

Mantieni i file danneggiati

Se abilitato, RAR non eliminerà i file che non sono stati estratti correttamente come, ad esempio, quando un archivio è danneggiato mentre, per definizione, RAR elimina questi file. In questo modo potete cercare di recuperare parte delle informazioni comprese nel file danneggiato.

Non estrarre i percorsi

Ignora le informazioni sui percorsi memorizzate ed estrae tutti i file archiviati in un unica cartella di destinazione.

Estrai gli archivi in sotto-cartelle

Questa opzione è disponibile solo se si sta cercando di scompattare più archivi contemporaneamente, mentre viene disabilitata se si sta gestendo un solo archivio. Selezionandola RAR pone il contenuto di ogni archivio in una propria sotto-cartella dove il nome è quello dell'archivio originale.

Visualizza i file estratti

Se l'estrazione viene completata con successo, con questa voce attiva al termine viene aperta la cartella dove sono stati collocati i file estratti.

Salva configurazione

Salva la configurazione di questo pannello come quella predefinita. Vengono salvate tutte le opzioni meno il percorso di destinazione. Lo stato dell'opzione "Estrai gli archivi in sotto-cartelle" viene salvato solo se avete selezionato più archivi. Le opzioni qui salvate sono utilizzate anche per personalizzare il menu contestuale e per controllare l'operazione di estrazione dagli Appunti.
