Conferma sostituzione file

Questo pannello viene visualizzato quando il file da estrarre o da copiare ha lo stesso nome di un file già esistente sul dispositivo.

Selezionate "Salta" per non considerare il nuovo file e mantenere quello già esistente.
Selezionate "Sostituisci" per sostituire il file esistente con quello nuovo.
"Salta tutto " e "Sostituisci tutto" eseguono le stesse operazioni di prima sia per il file corrente che per tutti i conflitti seguenti.

Se volete rinominare il nuovo file manualmente modificate il nome direttamente qui: il campo con il nome del file da creare è modificabile. Quando avete inserito il nuovo nome selezionate "Rinomina" o "Sostituisci". "Rinomina" e "Sostituisci" eseguono un'operazione similare con la sola differenza che "Rinomina" non permette di procedere se il nome originale e quello nuovo sono uguali. Se il nuovo nome file specificato corrisponde ad un file già esistente verrà visualizzato un'altra volta questo pannello.

Se selezionate "Rinomina automaticamente" tutti i nuovi file che andranno in conflitto con file già esistenti verranno rinominati con il formato "nome_file(N).estensione".

A seconda del tipo di comando l'opzione "Rinomina" può non essere disponibile.

Selezionate "Annulla" per terminare prematuramente l'intera operazione.
