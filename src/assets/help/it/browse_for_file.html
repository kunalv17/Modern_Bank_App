Sfoglia per file (archivio)

Questo pannello può ritornare o un nome file completo del percorso, o un percorso aggiornato o un nome di cartella.

Selezionate i nomi delle cartelle per aprirle, così potete trovare quella che contiene i file desiderati. Se trovate il file o l'archivio voluti, selezionate il loro nome.

Se volete selezionare solo il nome della cartella premete il pulsante "Solo cartella". Il comportamento può variare a seconda del pannello di RAR in cui ci si trova.
Quando viene invocato dal pannello di archiviazione questo pulsante modifica il percorso dell'archivio ma non il nome del file.
In altri pannelli di RAR esso ritorna il nome completo della cartella non preservando il nome originale.

Utilizzate il pannello di navigazione per selezionare la cartella corrente da un elenco di cartelle Preferite.

Una selezione prolungata del nome del file o della cartella vi permette di aprire il menu contestuale, dal quale potete attivare la creazione di una nuova cartella.
