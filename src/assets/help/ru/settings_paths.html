Папка распаковки по умолчанию

Указывает папку, предлагаемую по умолчанию для распакованных файлов в диалоге распаковки и контекстном меню.

Использовать папку по умолчанию

Указывает, когда применять папку распаковки по умолчанию. Возможные варианты: "Никогда", "Только внутри временной папки" и "Всегда". Все содержимое временной папки Android/data/com.rarlab.rar удаляется при удалении приложения RAR, и если она содержит ранее распакованные файлы, это может привести к потере данных. Поэтому по умочанию эта опция установлена в "Только внутри временной папки", и если архив хранится во временной папке, что может произойти при его передаче из другого приложения, RAR автоматически предлагает распаковать такой архив в папку распаковки по умолчанию вне временной папки.

Добавлять имя архива

Управляет добавлением имени архива без расширения к пути назначения в диалоге распаковки. Эта опция не влияет на команду распаковки в папку в контекстном меню.

История распаковки

Если включена, пути назначения в диалоге распаковки добавляются к истории и могут быть выбраны позже с помощью кнопки "Путь назначения" в этом диалоге. Пути, выбранные непосредственно из контекстного меню, не добавляются к истории вне зависимости от этой опции.

Стартовая папка

Выбор папки для отображения после запуска RAR. По умолчанию это папка верхнего уровня памяти устройства.
