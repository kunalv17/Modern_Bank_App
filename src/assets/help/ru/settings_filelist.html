Скрытые файлы

Если эта опция выключена, RAR не будет показывать файлы и папки, имена которых начинаются с точки. Эта опция влияет только на обычные файлы. Файлы и папки внутри архивов отображаются всегда, вне зависимости от наличия точки.

Миниатюры

Показывать уменьшенные изображения для APK файлов и графических файлов, таких как JPG. Только для обычных файлов, внутри архивов уменьшенные изображения не отображаются.
