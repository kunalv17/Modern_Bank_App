Сохранить файл

Нажимайте на имена папок, чтобы открыть их и выбрать папку для сохранения файла. Также вы можете редактировать имя файла. Нажмите "OK", чтобы сохранить файл в открытой сейчас папке.

Используйте панель навигации, чтобы выбрать текущую папку из списка избранных папок. Долгое нажатие на любом имени файла или папки открывает контекстное меню, которое позволяет создать новую папку.
