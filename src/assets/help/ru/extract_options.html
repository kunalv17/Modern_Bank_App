Параметры распаковки

Если распаковка запущена внутри архива, при просмотре содержимого архива, будут извлечены файлы и папки, выделенные в этом архиве. Если ничего не выделено, RAR автоматически выделит все содержимое архива.

Если распаковка запущена вне архива, при просмотре обычных неупакованных файлов, выделенные файлы рассматриваются как архивы и RAR попытается их распаковать. Если выделены папки, RAR будет искать в них и их вложенных папках файлы с архивными расширениями, такими как .rar, .zip, и подобными. Найденные файлы RAR попытается распаковать. Если ничего не выделено, RAR автоматически выделит все файлы с архивными расширениями в текущей папке.

Опции, доступные в этом каталоге, перечислены ниже.

Путь назначения

Имя папки для распакованных файлов. Если эта папка не существует, она будет создана автоматически. Вы можете принять предлагаемое по умолчанию имя папки, ввести другое имя папки, используя клавиатуру, или нажать кнопку "Обзор" для выбора папки назначения. Также вы можете нажать на кнопку "Путь назначения", чтобы быстро выбрать один из предлагаемых вариантов путей из списка. Подробности о списке предлагаемых путей доступны здесь.

Спрашивать перед перезаписью

Спрашивать перед перезаписью существующих файлов

Перезаписывать без запросов

Существующие файлы будут перезаписаны без каких-либо запросов

Пропускать существующие файлы

Не перезаписывать файлы, которые уже существуют.

Не удалять поврежденные файлы

Если эта опция включена, RAR не будет удалять файлы, которые не удалось корректно распаковать, например, из-за повреждения архива. По умолчанию RAR удаляет такие файлы. Иногда часть информации из поврежденного файла может быть извлечена.

Не распаковывать пути

Игнорировать пути файлов в архиве и распаковывать файлы из корневой папки архива и из вложенных папок в одну и ту же папку назначения.

Распаковывать архивы во вложенные папки

Эта опция доступна только при распаковке нескольких архивов и игнорируется для одного архива. Она помещает распакованное содержимое каждого архива в отдельную вложенную папку, имя которой генерируется на основе имени архива.

Показать распакованные файлы

Показать содержимое папки с распакованными файлами после успешного завершения распаковки.

Сохранить настройки

Сохранить текущее состояние опций в диалоге в качестве нового значения по умолчанию. Сохраняются все опции кроме пути назначения. "Распаковывать архивы во вложенные папки" сохраняется, только если вы выделили несколько архивов. Сохраненные здесь настройки также используются при распаковке с помощью буфера обмена и контексного меню.
