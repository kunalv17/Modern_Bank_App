Управление избранным

Нажмите на окошках метки или именах избранных элементов для выбора элемента. Вы можете выбрать один или несколько элементов. После завершения всех модификаций нажмите кнопку OK для их сохранения.

Доступны следующие команды:

Выбрать все

Выбрать все элементы в списке избранного. Если все элементы уже выбраны, эта команда снимет с них выделение.

Удалить

Удалить выбранные элементы.

Изменить

Изменить параметры выделенных элементов. Если выделено несколько элементов, редактироваться будет только первый из них.
