Параметры упаковки: Опции

Проверить упакованные файлы

Проверять файлы после упаковки. Эта опция особенно полезна, если также включено "Удалить файлы после упаковки". В этом случае файлы будут удалены, только если тест архива завершился успешно.

Создать отдельные архивы

Поместить каждый выделенный файл и папку в отдельный архив. Имена архивов генерируются из имени выделенного объекта и следующего за ним соответствующего расширения архива.

Использовать контрольную сумму BLAKE2

Целостность данных файлов в RAR архиве защищена контрольными суммами, вычисляемыми и хранящимися для каждого упакованного файла. По умолчанию, RAR использует контрольную сумму CRC32 длиной 32 бита. Если эта опция включена, вместо CRC32 используется 256-битная BLAKE2sp. Хотя CRC32 достаточно надежна для обнаружения почти всех ошибок, BLAKE2 обладает еще более высокой способностью детектирования ошибок. При использовании BLAKE2 может снизиться скорость и незначительно вырасти размер архива. Эта опция доступна только для архивов в формате RAR 5.0.

Показать время упаковки

Показать диалог, содержащий общее время упаковки, при успешном завершении архивирования. Может применяться для измерения производительности различных режимов упаковки.

Маска для имени архива

Добавить строку с текущей датой к имени архива при создании архива. Полезно для ежедневного резервного копирования. Строка с датой форматируется по указанной маске, которая может включать следующий символы:

Y год
M месяц
MMM  месяц как текстовая строка (Янв, Фев, и т.п.)
W номер недели (неделя начинается с понедельника)
A день недели (понедельник - 1, воскресенье - 7)
D день месяца
E день года
H часы
M минуты (обрабатываются как минуты, если расположены после часов)
I минуты (обрабатываются как минуты вне зависимости от позиции часов)
S секунды
N номер архива (числовое значение, начинающееся с 1).
При упаковке, RAR ищет уже существующий архив со сгенерированным именем, и если такой архив найден, увеличивает номер архива пока не получит незанятое имя. Символ формата 'N' не поддерживается при создании томов.

Каждый из символов строки формата, перечисленных выше, представляет только один символ, добавленный к имени архива. Например, используйте WW для двух цифр в номере недели или YYYY для четырехзначного года.

Если первый символ в строке формата это '+', позиции строки даты и основного имени архива меняются местами, так что дата будет вставлена перед именем архива.

Строка формата может содержать опциональный текст, заключенный в скобки '{' и '}'. Этот текст вставляется в имя архива.

Все прочие символы строки формата добавляются к имени архива без изменений.
