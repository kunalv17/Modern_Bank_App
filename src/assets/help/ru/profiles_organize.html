Управление профилями

Нажмите на окошке метки или имени профиля, чтобы выбрать профиль. Вы можете выбрать один или несколько профилей. После завершения всех модификаций нажмите кнопку OK для их сохранения.

Доступны следующие команды:

Выбрать все

Выбрать все профили упаковки в списке. Если все профили уже выбраны, эта команда снимет с них выделение.

Удалить

Удалить выбранные профили.

Установить по умолчанию

Назначить выбранному профилю статус 'по умолчанию'. Такой профиль будет использован для установки исходных параметров в диалоге упаковки. Если выбраны несколько профилей, статус 'по умолчанию' получит первый из них.
