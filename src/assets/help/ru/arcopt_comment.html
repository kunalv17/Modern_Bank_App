Параметры упаковки: Комментарий

Загрузить комментарий из файла

Укажите полное имя файла, чтобы добавить его содержимое как архивный комментарий. Такой файл должен иметь текстовый UTF-8 формат. Для выбора файла можно использовать кнопку "Обзор".

Ввести комментарий вручную

Введите текст, чтобы добавить его как архивный комментарий. Допускаются комментарии из нескольких строк.

Вы можете либо загрузить комментарий из файла, либо ввести его вручную. Использование обеих опций вместо невозможно.
