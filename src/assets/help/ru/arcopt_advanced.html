Параметры упаковки: Дополнительно

Сжатие

Доступны варианты "Без сжатия", "Самое быстрое", "Быстрое", "Стандартное", "Сильное" и "Максимальное". "Без сжатия" записывает файлы в архив на максимальной скорости без сжатия. Для каждого из следующих методов при переходе от "Самого быстрого" к "Максимальному" растет степень сжатия и снижается скорость. Это единственная опция, доступная для RAR и ZIP архивов на этой странице. Прочие опции предназначены только для RAR архивов.

Словарь

Размер области памяти для поиска и сжатия повторяющихся последовательностей в обрабатываемых данных. Больший словарь может улучшить степень сжатия больших файлов, особенно при создании непрерывного архива. Но больший словарь также уменьшает скорость сжатия и увеличивает расход памяти. Предлагаемые значения размера словаря зависят от размера свободной оперативной памяти. Если доступно больше памяти, вы сможете выбрать больший размер словаря. Максимальный размер словаря для формата RAR 4.x составляет 4 Мб.

Данные для восстановления

Добавить данные для восстановления, которые могут помочь команде "Восстановить архив" восстановить архив в случае повреждения. Размер данных для восстановления указывается в процентах от размера архива. Больший размер данных для восстановления позволяет восстановить поврежденную область большего размера, но увеличивает размер архива и требует больше времени при создании. Обычно 3 - 10% от размера архива являются оптимальным вариантом.

Размер тома

RAR позволяет создать архив, состоящий из нескольких файлов. Такой архив называется многотомным, а отдельные файлы, составляющие этот архив, называются томами. Здесь вы можете указать размер одного тома в многотомном архиве.

Пауза после каждого тома

Запрашивать подтверждение перед созданием каждого тома. Во время отображения этого подтверждения можно безопасно заменить носитель с уже созданным томом. Эта опция игнорируется, если размер тома не указан.

Томов для восстановления

Количество томов для восстановления в многотомном архиве. При распаковке архива каждый том для восстановления (с расширением .rev) позволяет восстановить один отсутствующий обычный том (с расширением .rar). Количество томов для восстановления должно быть меньше, чем количество обычных томов RAR и при необходимости оно уменьшается автоматически. Эта опция игнорируется, если размер тома не указан. Для восстановления отсутствующих обычных томов нужно поместить все имеющиеся .rar и .rev тома в одну и ту же папку. Также можно реконструировать отсутствующие тома без распаковки, просто выбрать один из .rev файлов в списке файлов и запустить команду "Восстановить архив" или нажать на имени .rev файла.
